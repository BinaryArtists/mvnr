## mvnr: recursive mvn command line tool

https://npmjs.org/package/mvnr

### Install:

    sudo npm install -g mvnr

### Usage:

    mvnr clean install -DskipTests

### Building:

    git clone git://github.com/kenglxn/mvnr.git
    cd mvnr
    npm test
